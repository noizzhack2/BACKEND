Purpose / מטרה:
This form is used to request reimbursement for meal and food expenses incurred during business travel, client entertainment, or team events.
טופס זה משמש לבקשת החזר הוצאות ארוחות ואוכל שנגרמו במהלך נסיעות עסקיות, בידור לקוחות או אירועי צוות.

Instructions / הנחיות:
1. Complete all required fields marked with an asterisk (*) / השלם את כל השדות הנדרשים המסומנים בכוכבית (*)
2. Attach itemized receipts for each meal / צרף קבלות מפורטות לכל ארוחה
3. Submit within 30 days of the expense / הגש תוך 30 יום מהוצאה
4. Expenses must comply with company travel policy / הוצאות חייבות לעמוד במדיניות נסיעות של החברה
5. Obtain manager approval before submission / קבל אישור מנהל לפני הגשה

Form Fields / שדות הטופס:

1. Employee Information / מידע עובד
   - Employee Name (required) / שם העובד (נדרש)
   - Employee ID (required) / מספר עובד (נדרש)
   - Department (required) / מחלקה (נדרשת)
   - Email Address (required) / כתובת דוא"ל (נדרשת)
   - Phone Number (required) / מספר טלפון (נדרש)
   - Cost Center (required) / מרכז עלויות (נדרש)

2. Expense Summary / סיכום הוצאות
   - Expense Report Period (required) - Date range picker / תקופת דוח הוצאות (נדרש) - בחירת טווח תאריכים
   - Report Submission Date (required) / תאריך הגשת הדוח (נדרש)
   - Currency (required) - Select: USD, EUR, GBP, Other / מטבע (נדרש) - בחר: דולר, יורו, פאונד, אחר
   - Total Meal Expenses (required) - Auto-calculated / סה"כ הוצאות ארוחות (נדרש) - מחושב באופן אוטומטי

3. Meal Details (Multiple entries allowed) / פרטי הארוחה (מותר מספר ערכים)
   For each meal / לכל ארוחה:
   - Date of Meal (required) - Format: MM/DD/YYYY / תאריך הארוחה (נדרש) - פורמט: MM/DD/YYYY
   - Restaurant/Vendor Name (required) / שם המסעדה/ספק (נדרש)
   - Location/City (required) / מיקום/עיר (נדרש)
   - Meal Type (required) - Select: Breakfast, Lunch, Dinner, Snacks / סוג הארוחה (נדרש) - בחר: ארוחת בוקר, ארוחת צהריים, ארוחת ערב, חטיפים
   - Business Purpose (required) - Select: Business Meeting, Client Entertainment, Team Event, Conference, Training, Travel / מטרה עסקית (נדרש) - בחר: ישיבת עסקים, בידור לקוח, אירוע צוות, כנס, הכשרה, נסיעה
   - Attendees Count (required) - Numeric value (minimum 1) / מספר המשתתפים (נדרש) - ערך מספרי (מינימום 1)
   - Attendee Names/Details (required) - Text area / שמות/פרטי המשתתפים (נדרש) - שדה טקסט
   - Meal Amount (required) - Currency format / סכום הארוחה (נדרש) - פורמט כספי
   - Tip Amount (optional) - Currency format / סכום תחזוקה (אופציונלי) - פורמט כספי
   - Tax Amount (required) - Currency format / סכום מס (נדרש) - פורמט כספי
   - Total for This Meal (required) - Auto-calculated / סה"כ לארוחה זו (נדרש) - מחושב באופן אוטומטי

4. Attendee Details / פרטי המשתתפים
   - Company Attendees (required) / משתתפים מהחברה (נדרש)
   - Client/External Attendees (required) / משתתפים חיצוניים/לקוחות (נדרש)
   - Business Discussion Topics (required) - Text area / נושאי דיון עסקיים (נדרש) - שדה טקסט
   - Expected Outcomes/Follow-ups (required) - Text area / תוצאות צפויות/המשך (נדרש) - שדה טקסט

5. Compliance / תאימות
   - All attendees are company employees or clients (required) - Checkbox / כל המשתתפים הם עובדי חברה או לקוחות (נדרש) - תיבת סימון
   - Expenses comply with company policy (required) - Checkbox / הוצאות תואמות את מדיניות החברה (נדרש) - תיבת סימון
   - Per Diem was not claimed for the same day (required) - Checkbox / דמי יום לא היו תובעים באותו יום (נדרש) - תיבת סימון
   - Duplicate reimbursement claim has not been submitted (required) - Checkbox / בקשת החזר כפולה לא הוגשה (נדרש) - תיבת סימון

6. Approvals / אישורים
   - Immediate Manager Name (required) / שם המנהל הישיר (נדרש)
   - Manager Email (required) / דוא"ל המנהל (נדרש)
   - Department Head Name (optional) - If amount exceeds threshold / שם ראש המחלקה (אופציונלי) - אם הסכום עולה על הסף
   - Approval Date - Read-only / תאריך אישור - קריאה בלבד
   - Approval Status - Read-only / סטטוס אישור - קריאה בלבד

7. Supporting Documentation / תיעוד תומך
   - Receipt Attachments (required) - Multiple files allowed / קבלות מצורפות (נדרש) - מספר קבצים מותר
   - Receipt Format Accepted - PDF, JPG, PNG, TIFF / פורמטים קבלות מקובלים - PDF, JPG, PNG, TIFF
   - Credit Card Statement (optional) - If applicable / הצהרת כרטיס אשראי (אופציונלי) - אם רלוונטי
   - Additional Documentation (optional) / תיעוד נוסף (אופציונלי)

8. Additional Information / מידע נוסף
   - Special Circumstances (optional) - Text area / נסיבות מיוחדות (אופציונלי) - שדה טקסט
   - Client Name/Company (optional) / שם לקוח/חברה (אופציונלי)
   - Project Code (optional) / קוד פרויקט (אופציונלי)
   - Funding Source (required) - Select: Department Budget, Project Budget, Client Recharge, Other / מקור מימון (נדרש) - בחר: תקציב מחלקה, תקציב פרויקט, חיוב לקוח, אחר

   - I confirm all information is accurate and truthful (required) - Checkbox / אני מאשר כי כל המידע מדויק ואמיתי (נדרש) - תיבת סימון
   - I have retained copies of all receipts (required) - Checkbox / שמרתי עותקים של כל הקבלות (נדרש) - תיבת סימון
   - All expenses are legitimate business expenses (required) - Checkbox / כל ההוצאות הן הוצאות עסקיות לגיטימיות (נדרש) - תיבת סימון

Important Notes / הערות חשובות:
- Standard meal limits per person: Breakfast $15, Lunch $25, Dinner $50 / מגבלות ארוחה סטנדרטיות לנפש: בוקר $15, צהריים $25, ערב $50
- Alcohol limited to 20% of meal cost / אלכוהול מוגבל ל-20% מעלות הארוחה
- Reimbursement processed within 5-10 business days / החזר מעובד תוך 5-10 ימי עסקים
- Missing itemized receipts may delay processing / קבלות מפורטות חסרות עלולות לעכב עיבוד
- Non-compliant expenses may be denied / הוצאות שאינן תואמות עלולות להיות נדחות
- Original receipts must show itemization, not just totals / קבלות מקוריות חייבות להראות פירוט, לא רק סכומים כוללים
- Hebrew and English requests are supported / בקשות בעברית ובאנגלית תומכות
