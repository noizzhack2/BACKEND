Fuel Expense Reimbursement Form
טופס החזר הוצאות דלק

Keywords / מילות מפתח:
fuel, gas, petrol, gasoline, diesel, station, pump, receipt, reimbursement, fuel expense, liters, gallons, price per liter, price per gallon, vehicle, car, license plate, odometer, חַדָּק, דלק, תחנת דלק, משאבה, קבלה, החזר, הוצאות דלק, ליטרים, מחיר לליטר, רכב, מכונית, מספר רכב

Purpose / מטרה:
This form is used to request reimbursement for fuel expenses incurred during business travel or authorized vehicle use.
טופס זה משמש לבקשת החזר הוצאות דלק שנגרמו במהלך נסיעות עסקיות או שימוש מורשה בכלי רכב.

Endpoint URL / כתובת קצה:
http://fuel_expense.com

Instructions / הנחיות:
1. Fill in all required fields marked with an asterisk (*) / מלא את כל השדות הנדרשים המסומנים בכוכבית (*)
2. Attach receipts or proof of fuel purchase / צרף קבלות או הוכחה לרכישת דלק
3. Submit within 30 days of the expense / הגש תוך 30 יום מהוצאה
4. Manager approval required before processing / אישור מנהל נדרש לפני עיבוד

Form Fields / שדות הטופס:

Only the following fields are required:
    - Date (required) / תאריך (נדרש)
       API Field Name: date
    - Location (required) / מיקום (נדרש)
       API Field Name: location
    - Car Number (required) / מספר רכב (נדרש)
       API Field Name: car_number
    - Fuel Station (required) / תחנת דלק (נדרש)
       API Field Name: station
    - Liters Amount (required) / כמות ליטרים (נדרש)
       API Field Name: liters_amount
    - Price Per Liter (required) / מחיר לליטר (נדרש)
       API Field Name: price_per_liter
    - Notes (optional) / הערות (אופציונלי)
       API Field Name: notes
    - Amount (required) / סכום (נדרש)
       API Field Name: amount

Purpose:
This form is used to request reimbursement for fuel expenses. Provide the following: date, location, car number, fuel station, liters amount, price per liter, notes, and total amount.

Instructions:
1. Complete all fields as specified (Notes optional).
2. Attach the fuel receipt.
3. Submit within 30 days of the expense.
