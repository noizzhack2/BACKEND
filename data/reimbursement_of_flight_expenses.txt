Reimbursement of Flight Expenses Form
טופס החזר הוצאות טיסה

Keywords / מילות מפתח:
flight, airfare, ticket, itinerary, airline, booking, boarding pass, seat, baggage, airport, travel, business trip, reimbursement, flight expense, כרטיס טיסה, חברת תעופה, הזמנה, עלות טיסה, החזר, טיסה, מסלול, שדה תעופה, נסיעת עסקים

Purpose / מטרה:
This form is used to request reimbursement for flight-related expenses incurred during business travel.
טופס זה משמש לבקשת החזר הוצאות טיסה שנגרמו במהלך נסיעות עסקיות.

Endpoint URL / כתובת קצה:
http://flight_expense.com

Instructions / הנחיות:
1. Complete all required fields marked with an asterisk (*) / השלם את כל השדות הנדרשים המסומנים בכוכבית (*).
2. Attach the flight receipt/ticket and boarding pass if available / צרף את קבלת הטיסה/כרטיס הטיסה וכרטיס העלייה למטוס אם זמין.
3. Submit within 30 days of the expense / הגש תוך 30 יום ממועד ההוצאה.

Form Fields / שדות הטופס:

Only the following fields are required:
   - Start Date (required) / תאריך התחלה (נדרש)
     API Field Name: startDate
   - End Date (required) / תאריך סיום (נדרש)
     API Field Name: endDate
   - Destination (required) / יעד (נדרש)
     API Field Name: destination
   - Source (required) / מקור (נדרש)
     API Field Name: source
   - Flight Number (required) / מספר טיסה (נדרש)
     API Field Name: flightNum
   - Amount (required) / סכום (נדרש)
     API Field Name: amount

Purpose:
This form is used to request reimbursement for flight expenses. Provide the following: start date, end date, destination, source, flight number, and total amount.

Instructions:
1. Complete all six fields as specified.
2. Attach valid flight receipt/ticket.
3. Submit within 30 days of the expense.
